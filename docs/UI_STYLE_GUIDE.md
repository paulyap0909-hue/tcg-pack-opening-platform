# UI Style Guide — Treasure Pack TCG

## Overall Style

The design should combine:

* Cyberpunk UI
* Futuristic HUD interface
* Web3 gaming landing page
* Gacha pack opening animation
* Neon card game dashboard
* Holographic card effects

The platform should feel like a premium gaming website, not an ecommerce store.

---

## Visual Keywords

* Dark futuristic
* Neon glow
* Holographic
* Glassmorphism
* Sci-fi panels
* Energy lines
* Animated gradients
* Pack reveal effects
* Game launcher layout
* Card vault interface

---

## Color Direction

Primary background:

* Deep black
* Dark navy
* Dark purple
* Space gradient

Accent colors:

* Electric blue
* Neon purple
* Gold
* Red
* Cyan

Pokémon Zone:

* Electric blue
* Yellow glow
* Energy-style animation

One Piece Zone:

* Dark navy
* Red
* Gold
* Treasure map texture
* Wanted poster inspired frame

---

## Typography Direction

Use strong gaming fonts for headings and clean modern fonts for body text.

Suggested style:

* Headline: futuristic, bold, wide spacing
* Body: clean, readable, modern
* Buttons: uppercase, glowing, compact

---

## Component Style

Buttons:

* Neon border
* Glow on hover
* Animated shimmer
* Strong CTA style

Cards:

* Holographic border
* Glass panel background
* Rarity badge
* Hover lift animation
* Glow based on rarity

Modals:

* Dark glass background
* Neon outline
* Animated entrance
* Clear action buttons

Pack Cards:

* Large cover image
* Remaining quantity
* Price in points
* Chase card preview
* Open Now button

Inventory Cards:

* Card image
* Rarity glow
* Condition badge
* Shipment status
* Favourite button

---

## Animation Direction

Use animation heavily but cleanly:

* Scroll reveal
* Parallax hero section
* Pack shake animation
* Card flip animation
* Glow burst
* Particle effect
* Rarity reveal
* Counter animation
* Leaderboard movement

Recommended libraries:

* GSAP
* Framer Motion
* Lottie
* Three.js only if needed

---

Homepage:

* Large cinematic hero
* Animated background

* How it works
* Live opening feed

Pack Lobby:

* Game store layout
* Category tabs
* Pack cards
* Limited drop banners

* Bottom: card pool / chase cards / terms

* Vault-style dashboard
* Filter by rarity / category / shipment status

Admin:

* Dark dashboard
* Sidebar
* Data cards
* Table management
* Charts
* Pack stock control

---

## Important Rule

Do not copy official Pokémon or One Piece artwork directly unless user provides licensed assets.

Use placeholder card designs, custom pack mockups and original visual direction for development.
